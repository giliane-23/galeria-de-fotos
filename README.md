# 📸 Galeria de Fotos

Projeto de uma galeria de imagens desenvolvido com foco em prática de **HTML, CSS e JavaScript**, com navegação entre páginas e exibição de fotos de forma organizada e interativa.

---

## 🚀 Acesse o projeto

🔗 https://giliane-23.github.io/galeria-de-fotos/

---

## 🖼️ Funcionalidades

* Página inicial com navegação
* Página de galeria com exibição de imagens
* Organização de layout com CSS
* Navegação entre páginas (HTML)
* Estrutura simples e funcional

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 📂 Estrutura do projeto

```
galeria-de-fotos/
│── index.html
│── galeria.html
│── stylegaleria.css
│── script.js
│── img/
```

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de:

* Praticar estruturação de páginas com HTML
* Trabalhar estilização com CSS
* Aprender navegação entre páginas
* Utilizar JavaScript para interações
* Publicar projeto com GitHub Pages

---

## 📚 Aprendizados

Durante o desenvolvimento, aprendi:

* Como organizar arquivos de um projeto front-end
* Como corrigir erros comuns (404, caminhos de arquivos)
* Como publicar um site com GitHub Pages
* Como fazer versionamento com Git
